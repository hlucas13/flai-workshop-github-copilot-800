## ğŸ¯ Overview

**OctoFit Tracker** is a comprehensive fitness tracking application designed for Mergington High School's physical education program. Created by PE teacher Paul Octo with the IT department led by Jessica Cat, this application helps students track their fitness activities, compete in friendly challenges, and stay motivated through gamification.

The application features a modern glassmorphism design with light/dark mode support, real-time activity tracking, team-based competitions, and personalized workout recommendations.

### The Story Behind OctoFit

Paul Octo, a physical education teacher with over 8 years of experience, noticed declining physical activity among students outside of PE classes. Inspired by a professional development conference on "Technology Integration in Physical Education," he envisioned an app that would:

- Make fitness tracking fun and engaging
- Create positive peer pressure through friendly competition
- Allow remote monitoring of student progress
- Provide personalized guidance based on individual fitness levels

Partnering with the school's IT department, Paul leveraged **GitHub Copilot agent mode** to bring OctoFit Tracker to life, focusing on educational aspects while AI handled technical implementation.

## ğŸš€ Getting Started

### Prerequisites

- Python 3.8+
- Node.js 18+
- MongoDB 6.0+
- Git

### Installation

#### 1. Clone the Repository

```bash
git clone https://github.com/hlucas13/flai-workshop-github-copilot-800.git
cd flai-workshop-github-copilot-800
```

#### 2. Backend Setup

```bash
# Navigate to backend directory
cd octofit-tracker/backend

# Create virtual environment
python3 -m venv venv

# Activate virtual environment
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Start MongoDB
# (Ensure MongoDB service is running)

# Run migrations
python manage.py migrate

# Populate database with sample data
python manage.py populate_db

# Start Django development server
python manage.py runserver 0.0.0.0:8000
```

#### 3. Frontend Setup

```bash
# Open new terminal
cd octofit-tracker/frontend

# Install dependencies
npm install

# Create environment file
echo "REACT_APP_CODESPACE_NAME=localhost" > .env

# Start React development server
npm start
```

#### 4. Access the Application

- **Frontend**: <http://localhost:3000>
- **Backend API**: <http://localhost:8000/api/>
- **Django Admin**: <http://localhost:8000/admin/>

2. Wait for environment to build
3. Ports 3000 (frontend), 8000 (backend), and 27017 (MongoDB) are auto-configured
4. Run setup commands as described above

### Endpoints

#### Users

- `GET /api/users/` - List all users
- `POST /api/users/` - Create new user
- `GET /api/users/{id}/` - Get user details
- `PATCH /api/users/{id}/` - Update user (partial)
- `PUT /api/users/{id}/` - Update user (full)
- `DELETE /api/users/{id}/` - Delete user

**User Model:**

```json
{
  "id": 1,
  "name": "John Doe",
  "email": "john@example.com",
  "team_id": "1",
  "created_at": "2026-02-13T10:00:00Z"
}
```

- `GET /api/teams/` - List all teams (includes member_count)
- `POST /api/teams/` - Create new team
- `GET /api/teams/{id}/` - Get team details
- `PATCH /api/teams/{id}/` - Update team
- `DELETE /api/teams/{id}/` - Delete team

